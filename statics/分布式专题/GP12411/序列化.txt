序列化
1什么是序列化 ，java是如何实现序列化的 
将java对象转换为字节序列的过程叫 java序列化
反序列化 java 字节数组转为对象的过程叫反序列化
java 是通过实现 java.io.Serializable该接口 实现序列化的
2 如果个子类实现了序列化，父类没有实现，那么父类中的成员变量能否被序列化？
不会 
如果是父类实现了序列化 子类没有实现 子类中的成员会被实例化
java 里面是父子传递的

5 有什么方法能够绕过transient的机制。这个实现机制的原理是什么？
通过深克隆绕过transient的机制
深克隆原理 
浅克隆是将对象的属性方法进行克隆但不包括类中对象的引用
深克隆是将对象的属性方法以及类中其他的对象进行复制 就是复制所有
